# Light Synchronization Problem

## সমস্যার বিবরণ

রংপুরের পীরগঞ্জের একটি মেলায় লাল এবং নীল রঙের দুটি বিশাল লাইট লাগানো হয়েছে।
- **লাল লাইট** প্রতি `x` সেকেন্ড পরপর জ্বলে
- **নীল লাইট** প্রতি `y` সেকেন্ড পরপর জ্বলে

রাত **৯:০০ টায়** দুটি একসাথে জ্বলেছিল। ঠিক `z` সেকেন্ড পরে দুটি একসাথে জ্বলবে কিনা বলতে হবে।

---

## Input Format
```
x y z
```

তিনটি পূর্ণসংখ্যা `x`, `y`, এবং `z`।

## Constraints
```
1 ≤ x, y, z ≤ 10^5
```

## Output Format

- যদি `z ` সেকেন্ডে দুটি লাইট একসাথে জ্বলে → `YES`
- অন্যথায় → `NO`

---

## Sample Input 0
```
10 15 20
```

## Sample Output 0
```
NO
```

## Sample Input 1
```
4 6 12
```

## Sample Output 1
```
YES
```

---

## Explanation

| x | y | z | LCM(x,y) | z % LCM == 0? | Output |
|---|---|---|----------|---------------|--------|
| 10 | 15 | 20 | 30 | 20 % 30 = 20 ≠ 0 | NO  |
| 4 | 6 | 12 | 12 | 12 % 12 = 0 | YES  |

> দুটি লাইট একসাথে জ্বলবে যখন `z`, `LCM(x, y)` দ্বারা বিভাজ্য হবে।
