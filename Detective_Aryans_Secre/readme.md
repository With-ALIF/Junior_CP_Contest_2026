# গোয়েন্দা আরিয়ানের সংকেত উদ্ধার

## সমস্যার বিবরণ

গোয়েন্দা আরিয়ান একটি চিরকুট পেয়েছেন যেখানে একটি রহস্যময় শব্দ S লেখা আছে। তিনি সন্দেহ করছেন এটি একটি গোপন সংকেত। সংকেতটি উদ্ধার করার জন্য তাকে শব্দটিকে একটি নির্দিষ্ট নিয়মে পরিবর্তন করতে হবে।

নিয়মটি হলো:

- ১. শব্দটির প্রতিটি Vowel (a, e, i, o, u) কে তার পরবর্তী ইংরেজি বর্ণ (Alphabet) দিয়ে প্রতিস্থাপন করতে হবে। (যেমন: 'a' হয়ে যাবে 'b', 'e' হয়ে যাবে 'f')।
- ২. শব্দটির প্রতিটি Consonant কে তার পূর্ববর্তী ইংরেজি বর্ণ দিয়ে প্রতিস্থাপন করতে হবে। (যেমন: 'b' হয়ে যাবে 'a', 'd' হয়ে যাবে 'c')।
- ৩. যদি বর্ণটি 'a' হয় (Vowel হিসেবে), তবে তা 'b' হবে। যদি 'z' হয় (Consonant হিসেবে), তবে তা 'y' হবে।

তোমাকে মূল শব্দটি দেওয়া হবে, তোমাকে পরিবর্তিত সংকেতটি প্রিন্ট করতে হবে। সকল ইনপুট Lowercase (ছোট হাতের অক্ষর) এ থাকবে এবং শুধুমাত্র ইংরেজি বর্ণ থাকবে।

- **Vowel** (a, e, i, o, u) → পরবর্তী বর্ণ (`a` → `b`, `e` → `f`)
- **Consonant** → পূর্ববর্তী বর্ণ (`b` → `a`, `z` → `y`)

---

## Input Format

প্রথম লাইনে একটি পূর্ণসংখ্যা N থাকবে (স্ট্রিংয়ের দৈর্ঘ্য)।

দ্বিতীয় লাইনে N দৈর্ঘ্যের একটি স্ট্রিং S থাকবে।

```
N
S
```

## Constraints
```
1 ≤ N ≤ 10⁵
S শুধুমাত্র lowercase (a−z)
```

## Output Format

পরিবর্তিত string প্রিন্ট করো।

---

## Sample Input 0
```
4
code
```

## Sample Output 0
```
bpcf
```

---

## Explanation

| অক্ষর | ধরন | পরিবর্তন | ফলাফল |
|-------|-----|----------|-------|
| c | Consonant | c → b | b |
| o | Vowel | o → p | p |
| d | Consonant | d → c | c |
| e | Vowel | e → f | f |

➡ `code` → `bpcf` ✅
