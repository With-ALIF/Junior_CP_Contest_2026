# Divisor Count Problem

## সমস্যার বিবরণ

তোমার ছোট ভাই গণিত শিখছে। সে তোমাকে একটি সংখ্যা দেয় এবং জানতে চায় সেই সংখ্যাটিকে মোট কতটি সংখ্যা দিয়ে নিঃশেষে ভাগ করা যায় (অর্থাৎ সংখ্যাটির মোট ভাজক কয়টি)। বড় সংখ্যা হলে সে আর হিসাব করতে পারে না। তুমি কি তাকে সাহায্য করার জন্য একটি প্রোগ্রাম লিখে দেবে?

যেমন: যদি সংখ্যাটি হয় `6`, তবে এর ভাজকগুলো হলো `1,2,3,6` । অর্থাৎ মোট ৪টি ভাজক আছে।

---

## Input Format
```
N
```

ইনপুটে একটি পূর্ণসংখ্যা N থাকবে।

## Constraints
```
1 ≤ N ≤ 10¹⁴
```

## Output Format

সংখ্যাটির মোট ভাজক সংখ্যা প্রিন্ট করো।

---

## Sample Input 0
```
16
```

## Sample Output 0
```
5
```

---

## Explanation

| সংখ্যা | ভাজকসমূহ | মোট |
|--------|----------|-----|
| 6 | 1, 2, 3, 6 | 4 |
| 16 | 1, 2, 4, 8, 16 | 5  |

> **কৌশল:** `1` থেকে `√N` পর্যন্ত লুপ চালালেই যথেষ্ট।
> প্রতিটি `i` যদি `N` কে ভাগ করে, তাহলে `i` এবং `N/i` দুটোই ভাজক।
> যদি `i == N/i` হয় (perfect square), তাহলে একবারই গণনা করো।

---

## Time Complexity
```
O(√N)
```

`N = 10¹⁴` হলে `√N ≈ 10⁷` — যা অনায়াসে চলবে।
