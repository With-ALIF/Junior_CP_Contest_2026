# Maximum Subarray Sum (Kadane's Algorithm)

## সমস্যার বিবরণ

ধরা যাক, তুমি শেয়ার বাজারে বিনিয়োগ করেছ। একটি নির্দিষ্ট সময়ের জন্য প্রতিদিনের লাভ বা ক্ষতির পরিমাণ একটি অ্যারে A তে দেওয়া আছে। পজিটিভ সংখ্যা মানে লাভ, আর নেগেটিভ সংখ্যা মানে ক্ষতি।

তোমাকে এমন একটি একটানা সময়কাল (Subarray) খুঁজে বের করতে হবে যেখানে তোমার সর্বমোট লাভের পরিমাণ সর্বোচ্চ হয়। তোমার কাজ হলো সেই সর্বোচ্চ যোগফলটি (Maximum Sum) খুঁজে বের করা।

---

## Input Format

- প্রথম লাইনে একটি পূর্ণসংখ্যা N (অ্যারের সাইজ) থাকবে।

- দ্বিতীয় লাইনে N টি পূর্ণসংখ্যা থাকবে (যা পজিটিভ বা নেগেটিভ হতে পারে)।

```
N
A[0] A[1] ... A[N-1]
```

## Constraints
```
1 ≤ N ≤ 10⁵
−10⁹ ≤ A[i] ≤ 10⁹
```

## Output Format

সর্বোচ্চ subarray যোগফল প্রিন্ট করো।

---

## Sample Input 0
```
8
-2 -3 4 -1 -2 1 5 -3
```

## Sample Output 0
```
7
```

---

## Explanation
```
অ্যারে:  -2  -3  [4  -1  -2  1  5]  -3
                  ↑___________________↑
                  যোগফল = 4-1-2+1+5 = 7 ✅
```

---

## Algorithm — Kadane's

> প্রতিটি index-এ দুটো সিদ্ধান্ত:
> - আগের subarray তে যোগ করব?
> - নাকি নতুন করে শুরু করব?
```
current = max(A[i], current + A[i])
max_sum = max(max_sum, current)
```
